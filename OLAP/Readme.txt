For this program we are using Car_Sales_Data_Set.csv as our data set, be sure your are hanving it in this directory

1. Execute the program and the program will generate three sorted file via Car_Sales_Data_Set.csv
2. Open the three generated csv file to know the reasult
3. Program ask you to enter a number in order to select corresponding cuboids
4. The result of selected cuboids will be shown on StdOut
5. Enter 0 to kill the program or keep entering other number
